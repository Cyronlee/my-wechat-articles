## 你好，公众号

这是一篇用来测试本地预览的示例文章。

### 为什么用 Markdown 写公众号？

用 Markdown 写作的好处很多：

- **排版一致**：不用在微信后台反复调整格式
- **版本管理**：所有文章都在本地，随时可以回溯修改
- **专注内容**：写作时只关注内容本身，样式交给渲染器

### 小结

这套工作流的核心思路是：

1. 用 Cursor Agent 根据需求撰写文章
2. 文章和图片都保存在本地 `articles/` 目录
3. 通过 doocs/md 编辑器预览最终的微信排版效果
4. 确认无误后复制到微信公众号后台发布
